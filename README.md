# 🥙 Шаурма Раннер

Telegram Web App игра в жанре idle/runner с элементами кликера и прокачки.

## 🎮 Как играть

1. **Клик по персонажу** — начать бежать
2. **Второй клик** — спринт (×2 скорость, ×2 расход стамины)
3. **Третий клик** — сесть отдохнуть (восстановление стамины)
4. **Четвёртый клик** — встать (цикл повторяется)

## 🌭 Киоски с шаурмой

Каждые N метров появляется киоск с шаурмой:
- 100м, 300м, 500м — первые три киоска
- Далее каждые 200м

За посещение киоска даётся ⭐ звезда улучшения.

## ⭐ Улучшения

- ❤️ **Запас стамины** +10
- ⚡ **Скорость** +0.5
- 🔄 **Регенерация** +0.2

## 🚀 Деплой на Vercel

1. Установи [Vercel CLI](https://vercel.com/docs/cli):
   ```bash
   npm i -g vercel
   ```

2. Задеплой проект:
   ```bash
   vercel
   ```

3. Для продакшн деплоя:
   ```bash
   vercel --prod
   ```

## 📱 Telegram Web App

Для использования в Telegram:

1. Задеплой проект на Vercel
2. Создай бота через [@BotFather](https://t.me/botfather)
3. Используй команду `/newapp` для создания Web App
4. Укажи URL твоего деплоя

## 🛠️ Технологии

- PixiJS 7.4.0 — рендеринг
- Telegram WebApp SDK — интеграция с Telegram
- localStorage — сохранение прогресса
